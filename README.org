#+TITLE: QPL Interpreter

* Prerequisites
- gcc

* Build
#+BEGIN_SRC shell
git clone git@git.ryanwatkins.me:base/sqint.git;
cd sqint;
make
#+END_SRC

* Usage
#+BEGIN_SRC shell
./bin/parse
#+END_SRC

* Description
Currently this implements some quantum gates and quantum adder operations as found in research papers.

* Example input(HADAMARD -> CNOT)
#+BEGIN_SRC
INITIALIZE R 2 0
U TENSOR HAD ID
APPLY U R
APPLY CNOT R
PEEK R RES
#+END_SRC
